Задачи

Что такое задача?

Задачи - классы, запускаемые с помощью командной строки или установленные в качестве задачи крон. В основном они используются для фоновых процессов, назначенных на определенное время задач и задач обслуживания. Задачи могут вызывать модели и другие классы, так же как контроллеры.

Создание задачи

В Fuel контроллеры размещены в каталоге fuel/app/tasks. Ниже дан пример задачи "пример": 

namespace Fuel\Tasks;

class Example
{

	public function run($message = 'Hello!')
	{
		echo $message;
	}
}

Ее можно вызвать через утилиту улучшения внутри oil:

$ php oil refine example "Good morning"
.

Когда в команде используется только имя задачи, делается ссылка на метод "run()".

Разбиение задач на дополнительные методы

Вы можете добавлять дополнительные методы в ваш класс задачи, чтобы разбить группу задач на более специфические задачи, которые можно вызывать отдельно.

public function current_date()
{
	echo date('Y-m-d');
}

После этого можно вызывать данный метод с помощью:

$ php oil refine example:current_date
