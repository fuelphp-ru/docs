Профилирование

Профилировщик приложений, входящий в FuelPHP, основан на быстром профилировщике PHP.

Что такое профилирование?

Профилировщик предоставляет информацию, относящуюся к профилированию и отладке, без необходимости добавлять много лишнего программного кода. Вам лишь нужно переключить один параметр конфигурации на истину, и вы получите доступ к автоматическому инструменту, обеспечивающему более быстрый и состоятельный анализ. Так как любой может использовать его, профилировщик также дает вам представление о том, где код опрережает анализ.

Профилировщик предоставляет вам интерфейс на основе вкладок, в котором можно найти следующую информацию:

  • Консоль. Это вкладка по умолчанию, дающая информацию об ошибках, записях журнала, использовании памяти или хронометраже выполнения.
  • Время загрузки. Это время загрузки запроса. Эта вкладка отображает подробные данные о выполнении.
  • База данных. Количество выполненных запросов, время выполнения, и в случае поддержки, анализ запроса.
  • Память. Общая пиковая память, использованная запросом.
  • Файлы. Полное уточнённое имя всех включенных файлов PHP и их размер.
  • Конфигурация. Содержимое хранилища конфигурации в конце запроса.
  • Сессия. Содержимое хранилища сессии в конце запроса.
  • GET. Содержимое массива $_GET.
  • POST. Содержимое массива $_POST.

Типовой экран профилировщика выглядит так:

В нижней части экрана профилировщика расположены три опции:

  • Низ. По умолчанию профилировщик накладывается. Нажмите данную опцию, если вы хотите увидеть весь вывод, профилировщик переместится за пределы конца вашего вывода. Нажмите снова, и он вернется обратно.
  • Высота. Переключает между большой (как на скриншоте) и маленькой панелью профилировщика.
  • Детали. Переключает детали профилировщика. Если она выключена, будет отображаться только панель вкладок.

Профилирование приложения

По умолчанию профилирование вашего приложения отключено. Вы можете настроить Fuel на запуск профилировщика через файл приложений config/config.php. 

'profiling'  => true,

После активации профилировщика он будет появляться внизу экрана браузера.

Профилирование базы даных

По умолчанию профилирование вашего соединения с базой даных отключено. Профилирование базы данных включается отдельно для каждого соединения путем установки в истину опции 'профилировщик' конфигурации соединения в файле config/db.php. 

'profiling'  => true,

Если вы испоьзуете несколько сред, убедитесь в правильности выбора файла конфигураций db.php.
