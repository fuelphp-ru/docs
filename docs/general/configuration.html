Конфигурирование вашего приложения

Введение

Среда разработки Fuel создана с учетом преобладания конфигурирования над правилами. В результате это дает высоконастраиваемую основу, на которой вы можете начать создавать свое приложение.

Файлы конфигурации хранятся внутри папки app/config. Базовая конфигурация приложения выполняется в app/config/config.php. Загруженная конфигурация может быть сделана связанной со средой, более подробно об этом в разделе Среды.

Опции конфигурации

Ключ Тип Значение по умолчанию Описание
base_url string (строка) 
null
 Базовый URL приложения. Может быть относительным. Должен содержать завершающий слэш. ('/foo/', 'http://example.com/')
url_suffix string 
''
 Любой суффикс, который должен добавляться к URL, генерируемым Fuel. Если суффикс является расширением, обязательно включите в него точку. ('.html')
index_file string 
'index.php'
 Имя главного файла самозагрузки. Установите его в false(ложь) или удалите, если вы используете mod_rewrite.
profiling bool (логический, булев) 
false
 Включать профилировщик или нет.
caching bool 
false
 Включать кэширование или нет.
cache_dir string 
APPPATH.'cache/'
 Каталог для сохранения файлов кэша. Этот каталог должен быть доступен для записи.
cache_lifetime int (целочисленный) 
3600
 Время существования кэша в секундах.
ob_callback callback (обратный вызов) 
null
 Обратный вызов, передаваемый ob_start(), установите в ob_gzhandler, чтобы включить кодирование gzip вывода.
errors.continue_on array (массив) 
array(
E_NOTICE,
E_WARNING,
E_DEPRECATED,
E_STRICT
)
 При каких ошибказ php продолжать выполнение.
errors.throttle int 
10
 Сколько ошибок показывать до прекращения их отображения. (предотвращает ошибки из-за нехватки памяти)
errors.notices bool 
true
 Показывать предупреждения или нет.
language string 
'en'
 Стандартный язык приложения, используемый классом Lang.
locale string 
'en_US'
 Используется для setlocale(), требуемой в некоторых установках php, установите в false, чтобы запретить ее установку.
Настройка синтаксиса для нее может отличаться для каждой ОС, например, Ubuntu требует суффикс .utf8 (кодировка).
encoding string 
'UTF-8'
 Стандартная кодировка приложений.
server_gmt_offset int 
0
 Смещение в секундах, смещение сервера от временной метки gmt при использовании time().
default_timezone string 
'UTC'
 Часовой пояс сервера.
log_threshold int 
Fuel::L_WARNING
 Порог записи в журнал. Ошибки серьезней какого уровня записывать в журнал. Ознакомьтесь с возможными значениями
log_path string 
APPPATH.'logs/'
 Каталог для сохранения журналов. Этот каталог должен быть доступен для записи.
log_date_format string 
'Y-m-d H:i:s'
 Формат даты/времени, используемый в журналах.
security.csrf_autoload bool 
false
 Проверять ли автоматически метку crsf. Прочитать больше о csrf.
security.csrf_token_key string 
'fuel_csrf_token'
 Какой ключ $_POST извлекать при проверке метки csrf.
security.csrf_expiration int 
0
 Устанавливает время окончания срока действия куки-файла csrf. Любое значение больше 0 - это число раделов, в которых срок действия куки-файла истечет.
security.uri_filter array 
array('htmlentities')
 Какие функции php использовать для фильтрации URI.
security.input_filter array 
array()
 Какие функции php должны фильтровать входные массивы ($_GET, $_POST и $_COOKIE). Может быть установлен в xxs_clean, это может вызвать скачок производительности исходя из размера ввода.
security.auto_encode_view_data bool 
true
 Кодировать (htmlentities) автоматически данные представления или нет.
security.whitelisted_classes array() 
array(
'stdClass',
'Fuel\\Core\\View',
'Fuel\\Core\\ViewModel',
'Closure'
)
 При включенном кодировании вывода все передаваемые объекты будут преобразовываться в строки или генерировать исключения, если они не являются экземплярами классом из данного массива.
cookie.expiration int 
0
 Количество секунд до того, как срок действия куки-файла истечет.
cookie.path string 
'/'
 Ограничивает путь, для которого доступен куки-файл.
cookie.domain string 
null
 Ограничивает домен, для которого доступен куки-файл.
cookie.secure bool 
false
 Передавать куки-файлы только через защищенные соединения.
cookie.http_only bool 
false
 Передавать куки-файлы только через HTTP, доступ через Javascript запрещен.
module_paths array 
array()
 Пути к каталогам модулей. Используется при добавлении модуля без указания местоположения.
always_load.packages array 
array()
 Which packages to autoload. Specify like so: array('package') or array('package' => PKGPATH.'path/to/package')
always_load.modules array 
array()
 Какие модулю загружаются автоматически. Укажите таким образом: array('package').Загруженные автоматически модули доступны через URL. Чтобы это работало, путь к модулям должен быть установлен в module_paths
always_load.classes array 
array()
 Какие классы загружаются автоматически и инициализируются.
always_load.config array 
array()
 Какие конфигурационные файлы загружаются автоматически. Загружайте конфигурационный файл в группу таким образом: array('config') или array('filename' => 'group'). Если вы не хотите загружать конфигурационный файл в группу, установите имя группы в null (пустой): array('filename' => null).
always_load.language array 
array()
 Какие файлы языков загружаются автоматически. Загружайте конфигурационный файл в группу таким образом: array('lang') или array('filename' => 'group'). Если вы не хотите загружать конфигурационный файл в группу, установите имя группы в null: array('filename' => null).
