Классы

Что такое класс?

Класс - это обычный класс PHP. Он не должен ничего расширять или соблюдать какие-либо соглашения, отличные от соглашений об именах, одинаковых для всех остальных классов в Fuel.

class Session

Данный класс будет загружен из файла app/classes/session.php.

Загрузка классов

В отличие от ряда других сред разработки, классы не нужно загружать вручную. Они будут автоматически загружаться, когда вы ссылаетесь на них в своем коде (контроллеры, модели, и т.д.).

Классы в подкаталоге

Точно так же как и контроллеры, имена классов должны быть написаны в нижнем регистре с первой буквой имени в верхнем регистре, в то время как подчеркивания в имени поместят класс в подкаталог.

Инициализация вашего класса

Можно заставить автозагрузчик Fuel выполнять определенные автоматические задачи сразу после загрузки класса, вроде тех, которые __construct() выполняет для экземпляров класса. Вы делаете это путем добавления открытого статического метода _init() в класс.

class Example {
	public static function _init()
	{
		// данный метод вызывается после загрузки класса
	}
}

Если загруженный класс имеет свой собственный метод _init(), и вместе с ним должен быть загружен родительский класс, то метод инициализации родительского класса не вызывается, если вы не вызываете его с помощью: parent::_init();
