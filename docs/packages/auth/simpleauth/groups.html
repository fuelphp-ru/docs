SimpleAuth Group

Как уже было сказано во введении, система аутентификации SimpleAuth поставляется с тремя встроенными драйверами, один из них SimpleAuth-Group.

Этот драйвер включается в файле настроек simpleauth. Он обеспечивает логику для проверки пользователя является ли тот членом группы, а также для получения информации о группе, и ролях заданных в группе.

Настройки

Настройки определяются в виде многомерного массива вида: 

array(
<id> => array(				// где <id> числовой идентификатор группы
'name' => 'name-of-this-group',		// имя группы, например 'Administrators' или 'Guests'
'roles' => array('role', 'otherrole'),	// список ролей
),
),

Файл настроек поставляется с Auth пакетом и содержит несколько примеров, которые можно использовать.

Методы

member($group, $user = null)

Позволяет проверить пренадлежит ли пользователь к группе.

Static No
Parameters
Param Default Description
$group required id группы
$user 
null
Идентификатор пользователя get_user_id(), или null если нужно проверить текущего пользователя.
Returns boolean. вернет true если юзер пренадлежит к группе, и false если нет.

get_name($group = null)

позволяет получить имя данного идентификатора группы

Static No
Parameters
Param Default Description
$group 
null
числовой идентификатор группы
Returns mixed. false если группы с таким id нет, или вернется имя группы.

Если не передать параметр, то вернется имя первой объявленной в настроках группы.

get_roles($group = null)

Извлекает роли по id группы.

Static No
Parameters
Param Default Description
$group 
null
The id of the group you want to get the roles of
Returns array. Если группы нет то вернется пустой массив, если не передать параметр то вернет роли первой обявленной в настройках группы.
