Auth пакет

Данный пакет представляет стандартыный интерфейс для авторизации в Fuel. Это позволяет нашим пользователям создавать свои собственные драйверы и легко интегрировать их с базовым ядром, сохраняя основные методы неизменными.

Введение

Создать Auth пакет, который бы отвечал запросам каждого - невозможно. Мы не хотим навязывать вам, какую-то определенную реализацию(driver) авторизации, мы помогаем создать свою собственную или быстро подключить и использовать чужую. Для этого и нужен Auth пакет. Также в Auth пакете имеется базовая реализация(driver) и для большинства её может хватить.

Интерфейс содержит 3 типа драйверов (и может расширятся другими драйверами): вход в систему(Login), группы(groups) и ACL(Access Control List — список контроля доступа, который определяет, кто или что может получать доступ к конкретному объекту). Login отвечает за вход/выход пользователя в систему и может работать с несколькими драйверами входа одновременно. Например, добавить к стандартному входу в систему возможность входа через Twitter. В качестве следующего шага можно сгруппировать эти 2 типа входа с драйвером groups, который будет отвечать за разделение пользователей на группы. И последнее, но не менее важное - вы можете использовать драйвер Acl, чтобы назначить определенные права этим группам.

Важно понимать, что выше приведен пример того, как можно использовать, а не, как вы должны использовать. Читайте в следующей секции, какие методы являются частью родного интерфейса: Basic Usage

Подключение

В конфиге требуется прописать

'packages' => array(
	'auth',
),

Настройки

Auth может быть настроен в config/auth.php файле и содержит следующие значения:

Param Type Default Description
driver string|array 
array('SimpleAuth')
 Драйвер авторизации.
verify_multiple_logins bool 
false
 Поддержка одновременного входа несколькимим способами.
salt string 
'put_your_salt_here'
 Ключ для хеширования паролей.

Файл настроек должен выглядеть как-то так:

<?php

return array(
	// Драйвер/Драйверы
	'driver' => array('SimpleAuth'),

	// Включение поддержки мультивхода под одним логином
	'verify_multiple_logins' => true,

	// Ключ для хеша
	'salt' => 'Th1s=mY0Wn_$@|+',
);

/* End of file auth.php */
