Пакет(Package)

Пакеты можно устанавливать и загружать вручную, но с помощью Oil этот процесс может проходить гораздо быстрее. Список пакетов находится тут fuel/core/config/package.php и как и любой конфигфайл он может редактироваться.

ДЛЯ КОРРЕКТНОЙ РАБОТЫ В ФАЙЛЕ php.ini НУЖНО РАСКОММЕНТИРОВАТЬ СТРОКУ extension=php_openssl.dll

Oil умная утилита, сначала она проверит установлен ли Git перед тем, как будет делать что либо с коммандой package. Если Git установлен, Oil установит пакет, как Git репозиторий вместо того, чтобы качать файлы напрямую. Git упростит обновление пакета, установку патчей и внесение других пользовательских изменений.

Комманда Install

$ php oil package install test-package
Downloading package: git://github.com/philsturgeon/fuel-test-package.git
remote: Counting objects: 13, done.
remote: Compressing objects: 100% (11/11), done.
remote: Total 13 (delta 3), reused 0 (delta 0)
Receiving objects: 100% (13/13), 10.85 KiB, done.
Resolving deltas: 100% (3/3), done.

Cloning into /Users/phil/Sites/fuel/fuel/packages/test-package...

Если же Git не установлен или выставлен флаг --direct будет скачан ZIP файл и распакован в fuel/packages/packagename. 

$ php oil package install test-package --direct
	Downloading package: http://github.com/philsturgeon/fuel-test-package/zipball/master

	DOCROOT/fuel/packages/test-package/README

	DOCROOT/fuel/packages/test-package/classes/belongsto.php
	DOCROOT/fuel/packages/test-package/classes/exception.php
	DOCROOT/fuel/packages/test-package/classes/hasmany.php
	DOCROOT/fuel/packages/test-package/classes/hasone.php
	DOCROOT/fuel/packages/test-package/classes/model.php

Комманда Un-install

$ php oil package uninstall test-package
Uninstalling package "test-package"

Замечание: В отличие от установки нет никакой разницы, установлен Git или нет, процесс удаления пройдет одинаково.
