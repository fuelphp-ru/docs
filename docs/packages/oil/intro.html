Если у вас Windows

Очень часто локальная разработка ведется на машинах под управлением ОС Windows, всвязи с чем могут возникнуть трудности при работе из коммандной строки. Чтобы комманды работали как нужно, запрос должен выглядеть так: 

$ cd v:/Sites/fuel
$ v:/usr/local/php5/php.exe oil -v
Fuel: 1.1

Oil утилита

Oil это специальная коммандная утилита, которая может быть использована несколькими способами, например для ускорения разработки приложения, а также его тестирования и запуска отложенных заданий (Tasks).

Oil является опциональной утилитой коммандной строки для ускорения разработки предоставляя несколько функций:

  • Generate - Создание MVC компонентов, миграций и скафолдинга.
  • Refine - Запуск отложенных заданий таких как миграция и своих собственных.
  • Package - Установка, апдейт и удаление пакетов.
  • Console - Тестирование вашего кода в режиме реально времени используя интерактивный шелл(shell - оболочка).

Каждая из этих комманд работает по своему достигая различных результатов, но перед тем, как ими воспользоваться лучше всего убедиться, что вы в нужной папке, а oil доступна и запущена. 

$ cd Sites/fuel
$ php oil -v
Fuel: 1.1

Помощь(Inline Help)

Oil содержит простую документацию внутри себя которая может быть вызвана коммандой:

$ php oil help

Usage:
  php oil [console|generate|help|test|package]

Runtime options:
  -f, [--force]    # Заменять файлы, которые уже существуют
  -s, [--skip]     # Пропускать файлы, которые уже существуют
  -q, [--quiet]    # Подавлять статус ответа

Description:
  The 'oil' command can be used in several ways to facilitate quick development, help with
  testing your application and for running Tasks.

Documentation:
  http://docs.fuelphp.com/packages/oil/intro.html

Каждая из этих команд имеет свой раздел справки, который можно найти, например введя:

$ php oil generate help
