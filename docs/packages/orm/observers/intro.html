Наблюдатели(Observers): Введение

Это система основанная на событиях позволяющая вам добавить своё поведение к событиям.

Добавление Observers к вашей модели

Вы можете добавить наблбдателя двумя способами: просто добавив имя наблюдателя вызываемого для всех событий или имя наблюдателя с массивом определенных событий для которых и будет вызываться наблюдатель.
Когда наблюдатель находится в том же пространстве имен, что и модель Вы можете не указывать "Observer_" префикс. Во всех остальных случаях вы должны обеспечить полное имя класса.

class Model_Article
{
	protected static $_observers = array(
		'example', // будет вызывать Observer_Example класс для всех событий
		'Orm\\Observer_CreatedOn' => array('before_insert'), // будет вызывать Orm\Observer_CreatedOn и только для события before_insert
	);
}
