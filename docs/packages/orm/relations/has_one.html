Отношение: Has One(Один к одному)

Задает отношение "один-к-одному" к другой модели. Целевая модель должна содержать "Belongs To" ссылку на текущую модель для поддержания обратной связи.

Примеры

protected static $_has_one = array('profile');

Ниже приведены примеры установления и разрыва has-one отношений:

// когда оба отношения новые:
$user = new Model_User();
$user->profile = new Model_Profile();
$user->save();

// когда оба отношения уже существуют
$user = Model_User::find(6);
$user->profile = Model_Profile::find(1);
$user->save();

// отключение отношений
$user = Model_User::find(6);
$user->profile = null;
$user->save();

Пример всех настроек, причем указаны дефолтные значения

// in a Model_User which has one profile
protected static $_has_one = array(
	'profile' => array(
		'key_from' => 'id',
		'model_to' => 'Model_Profile',
		'key_to' => 'user_id',
		'cascade_save' => true,
		'cascade_delete' => false,
	)
);
