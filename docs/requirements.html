Требования

Вебсервер
  • Любой вебсервер. Fuel был протестирован на Apache, IIS и Nginx.
  • Работает на всех основных операционных системах, включая *nix и Windows.
Версия PHP
  • Требуется версия 5.3 или выше.
Расширения PHP

Fuel использует функциональность, которую PHP определяет как "расширения". Чтобы избежать любых возможных проблем с совместимостью, Fuel использует только расширения, являющиеся частью кода PHP, как задокументировано на сайте PHP. Если используются расширения, входящие в состав репозитория PECL (который должен устанавливать вручную системный администратор), Fuel будет использовать их при их наличии, и решит проблему обходным способом при их отсутствии.

Ниже приведен список используемых расширений, и классы, которые используют их:

Расширение Используется в Комментарии
fileinfo Upload(пересылка файла) Документация.
Учтите, что на платформах Windows может понадобиться вручную установить DLL.
mbstring нет данных Документация.
Если оно отсутствует, среда разработки не поддерживает многобайтовый. Проверьте константу MBSTRING, чтобы узнать, подключена ли в вашей установке поддержка многобайтовых.
mcrypt Crypt(зашифрование-расшифрование) Документация.
Если оно отсутствует, PHPSecLib используется для имитации его функциональности.

Рекомендуемые требования

  • Вебсервер - Apache, работающий на *nix.
  • Версия PHP - 5.3.2
  • mod_rewrite - Чтобы убрать index.php из URL.
  • PHPUnit - 3.5.12
