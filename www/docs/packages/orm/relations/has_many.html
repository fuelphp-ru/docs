Отношение: Has Many(Имеет много)

Задает отношения "от одного-к-многим". Целевая модель должна содержать "Belongs To" ссылку на текущую модель для поддержания обратной связи.

Пример

protected static $_has_many = array('comments');

Ниже приведены примеры установления и разрыва has-many relations:

// когда оба отношения новые:
$post = new Model_Post();
$post->comments[] = new Model_Comment();
$post->save();

// когда оба отношения уже существуют
$post = Model_Post::find(1);
$post->comments[6] = Model_Comment::find(6); // assigning it to comments[6] is not required but recommended
$post->save();

// отключение отношений
$post = Model_Post::find(1);
unset($post->comments[6]);
$post->save();

Пример всех настроек, причем указаны дефолтные значения/h3> 
// в модели Model_Post которая has_many к comments
protected static $_has_many = array(
	'comments' => array(
		'key_from' => 'id',
		'model_to' => 'Model_Comment',
		'key_to' => 'post_id',
		'cascade_save' => true,
		'cascade_delete' => false,
	)
);
