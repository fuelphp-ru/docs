Refine(Улучшение)

Refine нужен для запуска отложенных заданий. "refine" потому, что Oil(масло) делается из Fuel(топливо). Вы можете прочесть больше об Tasks(Отложенные задания) и увидеть как их создавать и использовать.

Например task(задание) расположенное тут fuel/app/tasks/ с именем robots вызовет следующий результат.

$ php oil refine robots

			//\\  <\ _//^\\_ />  //\\

			      |   |   |   |
			      |---|   |---|

			      |   |   |   |
			      |___|   |___|
			      /   \   /   \

Не самое полезное задание, но пример хороший. Но вы можете использовать класс Cli для большей интерактивности или просто запускать задания для cron.
