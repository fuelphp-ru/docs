Auth_Acl_Driver

Этот драйвер является базовым классом для всех драйверов ACL. Это абстрактный класс, который содержит общие базовые методы для всех драйверов group, а также абстрактные методы, которые должны быть реализованы в своих собственых драйверах.

Базовые методы

Базовые методы уже реализованы и доступны через наследование. Эти методы позволяют взаимодействать с ACL драйверами и выполнять проверку доступа.

Те методы, которые есть, но не описаны в настоящей документации являются внутренними и не должны вызываться напрямую.

get_id()

Возвращает уникальный идентификатор драйвера. Это может быть использована для идентификации драйвера или выбора конкретного экземпляря драйвера.

Static No
Parameters None
Returns Строковый идентификатор.
Example 
// returns 'simpleacl'
$id = Auth::instance('simpleacl')->get_id();

set_config($key, $value)

Установка значений настроек.

Static No
Parameters
Param Default Description
$key required Configuration key name
$value required Value for this configuration key
Returns void
Example 
// set a config value on the simpleacl driver
Auth::instance('simpleacl')->set_config('key', 'value');

get_config($key, $default = null)

Получение настроек по ключу.

Static No
Parameters
Param Default Description
$key required Configuration key name
$default null Значение по умолчанию должно быть возвращено, если запрошенный ключ не существует
Returns mixed
Example 
// получает значение настройки simpleacl драйвера, если оно не существует верется false
$key = Auth::instance(simpleacl)->get_config('key', false);

Абстрактные методы

Все разрабатываемые ACL драйверы должны содержать в себе реализацию абстрактыных методов, либо их перегружать.

has_access($condition, Array $entity)

Метод has_access использует обявленные ACL драйверы для проверки доступа пользователей в соответствии с заданным условием.

Static No
Parameters
Param Default Description
$condition required задаёт необходимые права, строка или массив
$entity required идентификатор пользователя или групы вида array(driver_id, id).
Returns boolean, true если доступ разрешен, false если нет.
Example 
// проверка может ли пользователь добавлять комментарии
if ( ! Auth::instance('simpleacl')->has_access('comments.create'), array('simplegroup', 1))
{
Response::redirect('no_admin');
}
